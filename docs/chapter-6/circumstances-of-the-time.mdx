## Примеры

- sê roj berê - 3 дня назад
- du hefte berê - 2 недели назад
- sê roj bi şûn de - через 3 дня
- çar hefte bi şûn de - через 4 недели
- salek bi şûn de - через год
- duşema çûyî - в прошлый понедельник
- meha çûyî - в прошлом месяце
- sala çûyî - в прошлом году
- vê mehê - в этом месяце
- vê salê / îsal - в этом году
- vê hefteyê - на этой неделе
- meha bê - в следующием месяце
- sala bê - в следующием году
- hefteya bê - на слудующей неделе
- duşemê - в понедельник
- cara dawîn - в прошлый раз (последний)
- vê carê / îcar - в этот раз
- careke din - в следующий раз (другой)

## Дни недели (navên rojan)

|         |             |
| ------- | ----------- |
| yekşem  | воскресение |
| duşem   | понедельник |
| sêşem   | вторник     |
| çarşem  | среда       |
| pêncşem | четверг     |
| în      | пятница     |
| şemî    | суббота     |

### Примеры

- Îro çi roj e? - Сегодня какой день?
- Îro Sêşem e. - Сегодня вторник.
- Sibê roja çarşemê ye. - Завтра среда.
