## Положительные качества

|            |                                            |
| ---------- | ------------------------------------------ |
| dilovan    | нежный, сердечный                          |
| dilrehm    | милосердный, великодушный                  |
| dilgerm    | добрый, усердный, пылкий                   |
| dilpak     | искренний, чистосердечный                  |
| dilxweş    | радостный, веселый                         |
| dilsoz     | верный                                     |
| dilnizm    | смиренный                                  |
| dilvekirî  | искренний, откровенный (с открытым серцем) |
| destvekirî | щедрый                                     |
| mêrxas     | мужественный, храбрый                      |
| bextewar   | счастливый                                 |
| serbilind  | гордый (положительный оттенок)             |
| sernerm    | кроткий                                    |

### Примеры

- Yonatan mirovekî dilnizm bû û ew ji Yehowa re dilsoz bû. - Ионафан человеком смеренным был и Иегова верным был.
- Îsa Mesîh dilgerm bû, û mirovan ji sohbeta wî kêf distand. - Иисус Христос был добрый и люди от беседы его удовольствие получали.
- Peyva ku li Romayî 12:10 wek "dilovanî" hatiye tercûmekirin hezkirina di nav ferdên malbatê de terîf dike. - Слово которое в Римлянам 12:10 как "нежные чувства" переведено, любовь между членами семьи описывает.
- Çimkî Xudan gelek dilovan û tije rehm e. (Aqûb 5:11)
- Ji ber ku Xwedê dilrehm e, ew şaşiyên me efû dike. - Поскольку Бог милосердным есть, он ошибки наши прощает.
- Dilrehmiya Yehowa ji dilrehmiya dayikan hê jî mezintir e. - Милосердие Иеговы от милосердия матерей еще большим есть.
