Guh - м.р. ухо

Произношение: последнюю букву h практически не слышно;

:::warning ОСТОРОЖНО
Будьте внимательны т.к. есть похожее слово gû - нечистоты, экскременты.
:::

## Примеры

* Ewê guh bide.
* Ew guh dide.
* Wî guh da.
* Xwedê guh dide duayên te. - Бог слушает молитвы твои.
* Guh  bide şîreta Xwedê. - Слушайся совета Бога.
* Adem guh neda Xwedê. - Адам не послушался Бога.
