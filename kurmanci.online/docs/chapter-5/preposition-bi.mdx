---
title: Предлог "bi"
description: Предлог "bi"
---

1. “c”
2. создаёт творительный падеж (т.е. отвечает на вопрос кем? чем?)

## Примеры

* Melek her tim bi mirovan û bi mexseda Yehowa ... eleqedar dibin. - Ангелы всегда людьми и намерением Иеговы ... интересуются.
* Bi alîkariya prensîpên Kitêba Pîroz, hemû ferdên malbatê dikarin dilşad bin. - С помощью принципов Библии все члены семьи могут счастливыми быть.

* Bi alîkariyê - с помощью
* Bi navgîniyê - с помощью
* Bi destê...- (букв рукой...) с помощью
* Bi saya - благодаря
* Bi zedeyî - с излишком

* Dor bi dor - по очереди
