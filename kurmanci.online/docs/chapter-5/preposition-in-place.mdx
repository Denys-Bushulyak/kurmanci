|          |                 |
| -------- | --------------- |
| ser      | на, над         |
| bin      | под             |
| ber      | перед           |
| cem, rex | рядом           |
| nav      | между, внутри   |
| der      | снаружи, вне    |
| dû, pey  | вслед, за       |
| pêş      | вперед, впереди |
| paş      | позади, назад   |
| jêr      | низ, вниз       |
| jor      | верх, вверх     |

### Примеры

- Pişîk li ku derê ye? - Кот где?
- Pişîk li malê ye. - Кот в доме.
- Pişîk li ser kursî ye? Na. - Кот на стуле? Нет.
- Pişîk li bin maseyê ye? Na. - Кот под столом? Нет.
- Pişîk li ber neynikê ye? Na. - Кот перед зеркалом? Нет.
- Pişîk li cem (rex) dolapê ye? Na. - Кот рядом со шкафом? Нет.
- Pişîk di dolapê de ye? Erê. - Кот в шкафу? Да.

### Словосочетания с предлогом li

- Li gel - несмотря
- Li gor - в согласии
- Li hemberî - напротив, против
- Li dijî - против
- Li şûna - вместо
- Li derdora - вокруг
- Li derveyî - снаружи

---

- Li bajêr = di bajêr de = в городе
- Di nav bajêr de - внутри города
- Di hundirê bajêr de - (еще более внутри) в центре города
