## Примеры

В значении существуем:

* Ez hebûm - я был / существовал.
* Ez heme - я есть / существую.
* Ez hebim - я буду / буду существовать.

В значении у меня есть:

* Malbeta min hebû / heye / hebe - Семья моя (у меня) была / есть / будет.
* Malbeta min tunebû / tune / tunebe - Семьи у меня не было / нет / не будет.
* Zarokên min hebûn / hene / hebin - Дети мои (у меня) были / есть / будут.
* Li ba min malbet hebû / heye / hebe - У меня семья была / есть / будет.
* Li ba min zarok hebûn / hene / hebin - У меня дети были / есть / будут.
