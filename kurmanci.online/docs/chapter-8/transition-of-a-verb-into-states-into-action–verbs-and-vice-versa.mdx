|            Непереходные глаголы <br/> состояния             | ←→  |            Переходные глаголы <br/> действия            |
| :---------------------------------------------------------: | :-: | :-----------------------------------------------------: |
|                     **bûn**<br/>_быть_                      | ←→  |                 **kirin**<br/>_делать_                  |
|                 **hîn bûn**<br/>_обучаться_                 | ←→  |               **hîn kirin**<br/>_обучать_               |
|                **top bûn**<br/>_собираться_                 | ←→  |              **top kirin**<br/>_собирать_               |
|                 **aştî bûn**<br/>_мириться_                 | ←→  |               **aştî kirin**<br/>_мирить_               |
| **çêbûn**<br/>_появляться,<br/> готовиться,<br/> строиться_ | ←→  | **çekirin**<br/>_создавать,<br/>готовить,<br/> строить_ |
|     **sax bûn**<br/>_выздоравливать,<br/> поправляться_     | ←→  |               **sax kirin**<br/>_лечить_                |
|                                                             |  ←  |   **xwe sax kirin**<br/>_излечиваться (себя лечить)_    |
|    **azad bûn**<br/>_быть свободным, <br/>освобождаться_    | ←→  |            **azad kirin**<br/>_освобождать_             |
|                                                             |  ←  |         **xwe azad kirin**<br/>_освобождаться_          |
